# QSandbox

QSandbox - консольный конструктор нелинейных историй на Python (и мой первый проект на ЯП). 
Программа включает в себя и конструктор и "плеер".
Данные сохраняются/выгружаются из json.
Весь контент состоит из локаций, у каждой из которой есть:
- Имя (идентификатор, не отображается в игре)
- Заголовок
- Описание локации
- Переходы, которые состоят из:
    - Имени локации прибытия
    - Названия кнопки

Создание, редактирование локаций, а так же сама игра требует ввода команд с клавиатуры. Все инструкции выводятся перед запросом. 
Для запуска используйте menu.py. Модули builder.py и play.py можно так же запускать отдельно, для создания локаций и запуска игры соответственно.

Все названия чувствительны к регистру.

Конструктор устроен так, что можно создавать переходы в несуществующие локации, которые можно добавить позже. В конструкторе есть команда "проверка", запускающая поиск таких переходов.
